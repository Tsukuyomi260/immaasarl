# Images & Assets Structure

Mettre vos fichiers image ici:

## /logos
- Logo IMMAA (PNG/SVG)
- Logos partenaires

## /hero
- Image hero section (accueil)
- Banner images

## /services
- Images services (restauration, formation)
- Icons/illustrations

**Format recommandé:** PNG, SVG, WebP
**Tous les fichiers:** `/public/images/[dossier]/[fichier]`

**Utilisation dans les composants:**
```jsx
<img src="/images/logos/immaa-logo.png" alt="IMMAA Sarl" />
<img src="/images/hero/hero-main.jpg" alt="Restauration" />
```
